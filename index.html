WHPIC

What is it?

The WHPIC (Will Hodges Python Interpreted Calculator) is a proof of concept interpreter (sort of) written in Python. It is written using the Grako library for Python, which generates the parser.

How does it work?

The WHPIC uses an Expanded Backus-Naur Form grammar to define the rules of the calculator. Currently, the WHPIC is just a standard addition/subtraction/multiplication/division calculator that supports two number statements, ending with a semicolon.

The grammar is converted into a parser with Grako (see above). It generates calc.py, which has all the parser mumbo-jumbo.

The analysis of the content that is parsed occurs in imp.py. imp.py imports the parser and uses its functions to get a JSON-like result that is then parsed into numbers and operators, then calculated and returned.

imp.py is well-commented, so it should be basically self-explanatory.

Can I use it?

Sure! Just clone the repo, and read the Readme!

Can I contribute?

Yes, just clone the repo and improve it however you see necessary. Then, you can submit a pull request and have it merged with the main repo!
